## 介绍

使用 OpenCV 的 WeChatQRCode 模块与 PyZbar 解码库，识别图片中的二维码并返回解码结果。

## 使用方法

1. 发送命令：`/qrde`（与图片同一条消息或引用消息中发送）。
2. 插件将返回识别到的二维码文本；在 aiocqhttp（QQ 群）平台会以转发形式返回结果。
